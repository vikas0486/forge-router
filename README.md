# 🛠️ Forge Router

High-performance multi-LLM routing engine and interactive CLI.

## Installation

```bash
uv sync
```

## Usage

```bash
forge chat
```

## Commands

- `forge ask "question"`: Quick single question
- `forge chat`: Interactive chat mode
- `forge status`: Check provider availability
- `forge models`: List available models
- `forge doctor`: Diagnose configuration issues
